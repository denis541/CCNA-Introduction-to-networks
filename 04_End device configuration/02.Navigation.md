## ðŸ“˜ IOS CLI Primary Command Modes
Cisco IOS has several hierarchical modes. In this lab, I worked with the following:

| Mode | Prompt Example | Purpose |
|------|----------------|---------|
| **User EXEC Mode** | `Router>` | Basic monitoring, limited commands (e.g., `ping`, `show`). |
| **Privileged EXEC Mode** | `Router#` | Full access to device commands, including configuration and troubleshooting. |
| **Global Configuration Mode** | `Router(config)#` | Used to configure system-wide settings. |
| **Subconfiguration Modes** | `Router(config-if)#`, `Router(config-line)#` | Specific configuration for interfaces, lines, routing protocols, etc. |
